# Smart DataStudio 🚀

**Smart DataStudio** is a production-grade, self-contained data science workspace designed for seamless data exploration, automated anomaly detection, and machine learning/deep learning model training.

## ✨ Key Features

- **Full-State Persistence**: Save and load your entire workspace (datasets, dashboard layouts, and model configurations) using the custom `.sds` file format.
- **Smart Data Prep**: Automated detection of missing values, duplicates, and data anomalies with one-click cleaning.
- **ML Studio**: Interactive training of traditional ML models with real-time performance metrics and feature importance visualization.
- **DL Studio**: Professional-grade neural network builder with customizable architectures and inference testing.
- **Integrated Dashboard**: High-fidelity data visualizations using Recharts, adapting seamlessly to Dark and Light modes.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons, Recharts.
- **Backend**: Flask (Python), Pandas, NumPy, Scikit-Learn, TensorFlow/Keras.
- **Reporting**: High-resolution structured PDF generation engine.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- Python 3.9+

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/MuhammadAmaan178/Smart-Data-Studio.git
   cd Smart-Data-Studio
   ```

2. **Setup Backend**:
   ```bash
   cd backend
   pip install -r requirements.txt
   python app.py
   ```

3. **Setup Frontend**:
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
